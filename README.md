# Product Landing Page

##  Descripción

Este proyecto consiste en la creación de una landing page utilizando HTML5 semántico y buenas prácticas de accesibilidad. Forma parte del laboratorio 01 del nivel 201.

---

## Estructura Semántica

Se utilizaron etiquetas semánticas para organizar correctamente el contenido de la página:

* `<header>`: contiene la navegación principal
* `<nav>`: enlaces de navegación
* `<main>`: contenido principal
* `<section>`: agrupación de contenido por secciones
* `<article>`: contenido independiente dentro de una sección (FAQ)
* `<footer>`: información de contacto

Esto permite una mejor organización, legibilidad y mantenimiento del código.

---

##  Accesibilidad (A11y)

Se aplicaron principios básicos de accesibilidad:

* Uso de jerarquía correcta de encabezados (`<h1>`, `<h2>`, `<h3>`)
* Uso de atributos `alt` en imágenes (cuando se agreguen)
* Uso de `aria-label` en elementos interactivos
* Orden lógico del contenido para facilitar la navegación

---

##  Uso de IA

Se utilizó inteligencia artificial como apoyo para:

* Sugerir una estructura base de la landing page
* Validar el uso correcto de etiquetas semánticas
* Mejorar la organización del contenido

Las sugerencias fueron evaluadas y adaptadas según los requisitos del laboratorio.

---

##  Estilos

Se aplicaron estilos básicos con CSS para mejorar la presentación visual de la página, incluyendo:

* Tipografía
* Espaciado
* Colores en navegación y footer

---

##  Deploy

El proyecto fue publicado utilizando GitHub Pages:

*  Repositorio: https://github.com/raemiro/product-landing-page
*  Página web: https://raemiro.github.io/product-landing-page/

---

##  Conclusión

Este laboratorio permitió aplicar conceptos fundamentales de HTML semántico, accesibilidad y uso de herramientas como GitHub para el despliegue de proyectos web.
